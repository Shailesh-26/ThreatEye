# ThreatEye

ThreatEye is a Cyber Threat Hunting Platform designed to analyze security logs, detect threats, perform anomaly detection, visualize attacks, and provide SOC-style monitoring dashboards.

## Tech Stack

### Frontend
- React
- TailwindCSS
- Recharts

### Backend
- FastAPI
- Python

### Database
- MongoDB

### Machine Learning
- Scikit-Learn
- Pandas
- NumPy

### Deployment
- Docker
- GitHub

## Features

- JWT Authentication
- Log Upload & Management
- Brute Force Detection
- Port Scan Detection
- DDoS Detection
- Geo-IP Analysis
- Threat Intelligence Feed
- MITRE ATT&CK Mapping
- Threat Timeline
- PDF Reports
- ML-based Anomaly Detection
